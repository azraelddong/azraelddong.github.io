网球项目文档

网球学练场项目运行文档

[!NOTE]

参考地址：http://oa.infoepoch.com/issue/34040

项目下载

  • 依赖框架：http://gitlab.infoepoch.com/rdd/wqxlg-framework
  • 后端项目：http://gitlab.infoepoch.com/rdd/wqxlg-admin
  • 后管前端项目：http://gitlab.infoepoch.com/rdd/wqxlg-admin-spa
  • 小程序项目：http://gitlab.infoepoch.com/rdd/wqxlg-weapp-spa

项目启动

后端项目启动

版本依赖

  • jdk17+

  • maven3.8+

  • 配置环境变量：WQXLG_APP_SECRET=60157aZbN1w8Xu8VF3oSNZXGrKw3jZ/3b/bTL41BS4FlvaGcudrpl1Q57meWfYuSTym92SnLwuw=;WQXLG_WORKER_FOLDER=后端项目根目录路径地址

  • 本地测试环境项目启动配置

后管前端项目启动

依赖版本：

  • node18+
  • npm9.8.1+

项目启动：

Package.json文件中定义的启动脚本命令:

npm run dev：测试环境启动（本地启动测试常用⭐️）

npm run prod：生产环境启动

npm run build：生产环境打包

npm run build:dev：测试环境打包

小程序项目启动

依赖版本：

  • node18+
  • npm9.8.1+

[!IMPORTANT]

有两个文件需要注意：

.env.development：本地环境变量文件，需要将启动地址改成本机ip地址

manifest.json：小程序配置文件；

项目启动：

控制台启动命令：

本地测试启动：npm run dev:mp-weixin

生产打包：npm run build:mp-weixin

使用npm run dev:mp-weixin命令启动成功后，打包微信开发者工具，打开编译打包后的目录

微信开发者工具导入编译打包后的目录路径，点击创建

微信开发者工具配置修改成下面一样，点击清缓存，重新编译刷新
